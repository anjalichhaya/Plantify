# Plantify - Indoor Plant E-commerce Website
A beautiful and responsive e-commerce website for buying indoor plants, pots, and trays. Built with pure HTML, CSS, and JavaScript with a modern dark theme and user authentication.

# Features
-Authentication System
Email validation (must contain @)
Password strength check (minimum 6 characters)
Session management using LocalStorage
Secure logout functionality

# E-commerce Features
-Product Catalog
Indoor Plants 
Decorative Pots 
Plant Trays 

# User Interface
Modern dark theme (#121212, #1f1f1f)
Collapsible sidebar navigation
Hover effects
Clean and minimal design
